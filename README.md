# Presskit - Developer Portfolio

Portfolio personal con intro animada, video de fondo y diseño moderno dark navy.

## 🚀 Tecnologías

- HTML5
- CSS3
- JavaScript (ES6+)
- Vite

## 📦 Instalación

```bash
npm install
```

## 🛠️ Desarrollo

```bash
npm run dev
```

El servidor se iniciará en `http://localhost:5173`

## 🏗️ Build para Producción

```bash
npm run build
```

Los archivos optimizados se generarán en la carpeta `dist/`

## 📤 Deploy en GitHub Pages

1. Build del proyecto:
```bash
npm run build
```

2. Subir la carpeta `dist/` a GitHub Pages o configurar GitHub Actions para deploy automático.

## 📁 Estructura

```
presskit-programacion/
├── index.html          # Página principal
├── src/
│   ├── main.js        # Entry point
│   ├── intro.js       # Lógica de intro

│   └── style.css      # Estilos principales
├── public/

```
